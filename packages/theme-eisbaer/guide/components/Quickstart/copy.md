
Install the library.

```
$ yarn add -D @reuters-graphics/style-theme-eisbaer
```

Import the main stylesheet in your code.

```scss
// Import in SCSS
@import "~@reuters-graphics/style-theme-eisbaer/scss/main";
```

```js
// Import in JS
import 'reuters-graphics/style-theme-eisbaer/scss/main.scss';
```
